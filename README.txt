The containing files :
----------------------
• Class Diagram : a photo that discribes the class diagram
• GoFo : the program code 
• Interface Designs : screenshots of the actual application
• JavaDoc : an explanation of all the details of the code
• SequenceDiagrams : photos that describe the sequence diagrams

The used tools :
-----------------
• StarUML : Used to draw the Class Diagram.
• Lucidchart : Used to Draw the Sequence Diagrams.
• Figma : Website used to draw the UI/UX Designs.
• Eclipse : Software Used in Implementation for the code in Java Language and the JavaDoc.
